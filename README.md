# Phone Catalog
A responsive e-commerce product catalog built with React and TypeScript.

The application allows users to browse mobile phones, tablets, and accessories, view detailed product information, search and sort products, add items to favorites, and manage a shopping cart.

## Features
- Responsive layout for mobile, tablet, and desktop devices
- Home page with promotional banners and product sliders
- Separate product pages for phones, tablets, and accessories
- Product sorting by newest, alphabetically, and price
- Product search using URL query parameters
- Pagination with configurable items per page
- Product details page with images, colors, capacities, and specifications
- Suggested products section
- Shopping cart with quantity controls
- Automatic cart quantity and total calculation
- Favorites page with saved products
- Cart and favorites persistence using Local Storage
- Breadcrumb navigation
- Loading and error states
- Product-not-found page
- Page-not-found page
- Smooth hover effects
- Responsive navigation
- Back-to-top functionality

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/aholubko/phone-catalog.git
cd phone-catalog
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the project locally

```bash
npm run dev
```

Open the local address displayed in the terminal.

## Available Scripts

Run the development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Run the linter:

```bash
npm run lint
```

Deploy the project to GitHub Pages:

```bash
npm run deploy
```
